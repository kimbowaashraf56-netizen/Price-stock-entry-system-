# Price & Stock Entry System

A simple, lightweight web application for tracking inventory prices and stock quantities. Perfect for small businesses, warehouses, and inventory management.

## Features

✨ **Easy Data Entry** - Add items with brand, quantity, price, and date
📊 **Live Table Display** - View all entries with automatic totals
💰 **Auto Calculations** - Automatic total calculation (Quantity × Price)
📈 **Grand Total** - Running sum of all transaction values
📥 **CSV Export** - Download data for Excel/spreadsheets
🗑️ **Clear All** - Reset entire dataset with confirmation
📱 **Responsive Design** - Works on desktop, tablet, and mobile
📅 **Date Picker** - Defaults to today's date for convenience

## How to Use

1. **Open** `index.html` in your web browser
2. **Fill in the form:**
   - Item name (e.g., Cement, Iron Rods)
   - Brand name (e.g., Hima, Blue Band)
   - Quantity
   - Price per unit (in UGX)
   - Date
3. **Click "+ Add Entry"** to add to the table
4. **Export as CSV** to download for Excel
5. **Clear All** to reset data (with confirmation)
